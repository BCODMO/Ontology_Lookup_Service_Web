Ontology Lookup Service roadmap

This new version of OLS is a complete re-write of the code and includes some significant changes to how we index and serve up ontology data. This includes an all new API for programmatic access to the data. The old OLS service is no longer able to update the ontologies and we plan to deprecate the service entirely in 2016. This document outline the road map for releasing the new OLS service and provides a guide for migrating from the old SOAP API to the new REST API.

Timeline

  • November 1st 2015
    • OLS beta announced running at http://www.ebi.ac.uk/ols/beta
    • Announcement of deprecation of old OLS services running at http://www.ebi.ac.uk/ontology-lookup (also redirects from http://www.ebi.ac.uk/ols)
    • Freeze data in old OLS and stop nightly update of ontologies
  • January 31st 2016
    • We will start an instance of the old OLS at http://www.ebi.ac.uk/ontology-lookup/v2. This will serve the old services and data until October 1st 2016
  • March 1st 2016
    • We will move new OLS to production and serve from http://www.ebi.ac.uk/ols
    • http://www.ebi.ac.uk/ontology-lookup will redirect to http://www.ebi.ac.uk/ols
    • http://www.ebi.ac.uk/ols/beta will redirect to http://www.ebi.ac.uk/ols
    • The old OLS service will be archived and only available from http://www.ebi.ac.uk/ols/v2
  • October 1st 2016
    • We will stop redirecting request from http://www.ebi.ac.uk/ols/beta
    • We will shutdown the old OLS service running at http://www.ebi.ac.uk/ols/v2

What will be deprecated

  • As of March 1st 2016 we will no longer provide a MySQL dump of OLS
  • As of March 1st 2016 the SOAP API will no longer be accessible from http://www.ebi.ac.uk/ontology-lookup/OntologyQuery and only available from http://www.ebi.ac.uk/ols/v2/OntologyQuery until October 1st 2016
