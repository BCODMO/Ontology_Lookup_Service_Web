Ontology Lookup Service

Welcome to the EMBL-EBI Ontology Lookup Service (OLS). OLS provides a web service interface to query multiple ontologies from a single location with a unified output format. The OLS can integrate any ontology available in the OBO, OWL or SKOS format.

This is a beta site under active development and is primarily aimed at testing functionality of the all new OLS REST API. Please contact us with any feedback or suggestions.
