Term overview tag

This tag can be used if you have an ontologie term label and want to get an overview of it as shown below.

Integrating it in your code as a jquery plugin

If you need to learn about jquery plugin, go here.
If you want to have a look at the jquery plugin registry, go here

For that, just import the following files in the head tag of your html page :
  • the ontology.css file
  • the jquery.min.1.11.2.js javascript file
  • the termOverview.js javascript file
  • the jquery.mCustomScrollbar.concat.min.js javascript file

Inside the body of your html page, add any tag (div or p) with the following attributes :
  • data-label : this is the label of the term you're interested in (MANDATORY).
  • data-ontology-name : this is the name of the ontology you want to retrieve the term from (MANDATORY)
  • data-json-path : this is the path to the solr server where your ontologies are indexed. If not given it will default to the EBI solr server. If the url is given, it should contain the string "LABEL_TO_REPLACE" where the given label should be inserted and the string ONTOLOGY_NAME_TO_REPLACE where the ontology name should be inserted. (NON MANDATORY)
Then in a script tag add the following code :
jQuery(document).ready(function () {
jQuery.noConflict();
jQuery('p').injectTermOverview();
});

Your page should look like :


<html>
<head>
<link rel="stylesheet" href="css/ontology.css">
<script src="script/jquery.min.1.11.2.js" ></script>
<script src="script/termOverview.js"></script>
<script src="script/jquery.mCustomScrollbar.concat.min.js" ></script>
<script>
jQuery(document).ready(function () {
jQuery.noConflict();
jQuery('p').injectTermOverview();
});
</script>
</head>
<body>
<p data-label="insomnia" data-ontology-name="EFO" data-json-path="http://localhost:8983/solr/ontology/select?q=label%3ALABEL_TO_REPLACE+AND+ontology_name%3AONTOLOGY_NAME_TO_REPLACE&rows=1&wt=json&indent=true"></p>
</body>
</html>

Integrating it just by adding a div tag in your html page


For that, just import the following files in the head tag of your html page :
  • the ontology.css file
  • the jquery.min.1.11.2.js javascript file
  • the termOverview.js javascript file
  • the jquery.mCustomScrollbar.concat.min.js javascript file

and inside the body tag of your html page add one of the following tag :

<term-overview json-path="http://localhost:8983/solr/ontology/select?q=label%3ALABEL_TO_REPLACE&rows=1&wt=json&indent=true" ontology-name="EFO" label="lung"></term-overview>
The json_path attribute is the url pointing to your solr server where your ontology/ontologies is/are indexed, if you don't have one just don't put the json_path attribute and the EBI default one will be used.
The ontology-name attribute is the name of the name of the ontology for which you want solr to return results.
The label is the label of the term you want to overview.

Your overall page should therefore look like :

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="css/ontology.css">
<script src="script/jquery.min.1.11.2.js" ></script>
<script src="script/termOverview.js"></script>
</head>
<body>
<term-overview json-path="http://localhost:8983/solr/ontology/select?q=label%3ALABEL_TO_REPLACE&rows=1&wt=json&indent=true" ontology-name="EFO" label="lung"></term-overview>
</body>
